Print Cards for Jira

Overview of features and usage

Introduction

This app allows the user to create a printable representation of Jira workitems.

How to use

    1. Create a filter to select your workitems
    2. Select Apps > Print Cards
    3. Select custom fields
    4. Generate the HTML report and print it
